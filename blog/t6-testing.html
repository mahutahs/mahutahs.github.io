    In the javascript language, describe all the use cases for each of: parentheses () - e.g. '1. Execute a function var result = doSomething()' brackets [] braces {} single quotes '' double quotes ""

    "Hmm.. thats an interesting one.. "

    Parentheses ()

    These are used by functions and methods. They generally hold input information, such as somethingto print on the console with console.log()

    Brackets []

    Brackets are like parentheses as they are like containers used to store information. This can hold many things such as numbers or strings("sentences"). Arrays which store alot of information typically brackets.

    Braces {}

    These hold commands for a chunk of control statemements and functions but are also used to define objects.

    Quotation Marks "" ''

    These are used to define strings and can be interchangeable thus having strings in strings. Strings are essentially sentences or put into blog posts like this one :)
